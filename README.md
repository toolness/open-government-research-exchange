## Quick Start

From a terminal, run:

```
npm install
npm start
```

## Deploying to S3

First, set the following environment variables:

* `AWS_ACCESS_KEY_ID` should be set to your AWS access key.
* `AWS_SECRET_ACCESS_KEY` should be set to your AWS secret access key.

Then run `npm run s3`.
